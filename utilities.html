Utilities

Make sure the A Dragonborn's Fate profile is active before proceeding.

Skyrim Script Extender

Installation instructions:

  1. Click the download link next to Current Aniversary Edition build 2.1.5 (game version 1.6.353)
Make sure you are downloaded the Anniversary Edition version, not the Classic, Special Edition, or VR versions.
  1. Extract the contents of the skse64_2_01_05 folder from the downloaded archive to the game's Root folder
Make sure you are extracting all of the files inside of the skse64_2_01_05 folder, not just the folder itself.
If you do not know what the Root folder is, read the Key Terminology section from the Initial Setup page.

- Extends the scripting capabilities of the game

In order for the script extender to work, you need to launch the game using the skse64_loader.exe file. It will automatically be added to Mod Organizer 2 upon restarting it.

SSE Engine Fixes

Installation instructions:

  1. Download and install the Main File - (Part 1) SSE Engine Fixes for 1.6.xxx normally through MO2
  2. Download the Main File - (Part 2) Engine Fixes - skse64 Preloader and TBB Lib file using the Manual Download button
  3. From the downloaded archive, extract all the files to the game's Root folder
  4. Once the files are installed, right-click on the SSE Engine Fixes mod in the left pane of MO2 and select Open in Explorer
  5. Navigate to SKSE\Plugins and open EngineFixes.toml in a text editor (such as notepad)
  6. Set RegularQuicksaves to true (line 13)
  7. Set SleepWaitTime to true (line 18)

- Fixes numerous bugs within the game's engine

powerofthree's Papyrus Extender

Installation instructions:

  1. Download the main file and begin the installation normally through MO2
  2. You will be prompted with a FOMOD installer, in which you should just click Install

- Adds new scripting functions for mods to take advantage of


REMINDER: You need to run the game through the SKSE option in Mod Organizer 2 from now on, NOT the Skyrim Special Edition option. You may need to restart MO2 for the option to appear.
