Visuals

There may be less mods than you may be used to with a modded game, but there is good reason for that. Skyrim Special Edition already has beautiful visuals in my opinion. Any visual improvements you make to the game also come at a cost. ENBs may make the game look slightly better, but they can also tank performance depending on the preset. In a game that already has dynamic shadows, ambient occlusion, god-rays, and more features that ENB add, I do not see them as necessary. Texture mods may improve some textures, but they can also cause performance drops and stuttering. They also create a great disparity between the new textures and the vanilla textures, both in quality and in style, making for some jarring texture transitions or seams. Overall, I do not think these types of mods are worth it.

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select All Mods -> Create Separator
  2. Name the separator Visuals

Security Overhaul SKSE - Lock Variations

Installation instructions:

  1. Download the main file through MO2 and begin the installation normally
  2. You will be prompted with a FOMOD installer, in which you should just select the 2K Textures option then select Install
If you have a very weak PC, you can select the 1K Textures instead.
  1. Additionally install Security Overhaul SKSE - Add-ons and Security Overhaul SKSE - Regional Locks, selecting the same texture size options as you selected in the first FOMOD

- Adds variety to the lockpick interface based on door/container type

Blended Roads

Installation instructions:

  1. Download the main file through MO2 and begin the installation normally
  2. You will be prompted with a FOMOD installer, in which you should just select the REALly Blended Roads option from the top section
If you have a weaker PC, you can select the regular Blended Roads option to avoid any performance decrease
  1. Finish the installation normally

- Better blends roads into the landscape textures to avoid seams and gives them 3D meshes to look more natural

Cathedral Water Overhaul

- Improves and fixes many aspects of SSE's water

Enhanced Lights and FX

Installation instructions:

  1. Download the main file through MO2 and begin the installation normally
  2. You will be prompted with a FOMOD installer, in which you should just select ELFX Enhancer on the first page
If you have a weaker PC, or just don't like the darker interiors, you can simply just click Next until the installer finishes without selecting any additional options.
  1. Finish the installation normally
  2. Additionally install ELFX Fixes (just select Install in the FOMOD)

- Greatly improves interior lighting/light sources

Dimmer Fog

Installation instructions:

  1. Download the main file through MO2 and begin the installation normally
  2. You will be prompted with a FOMOD installer, in which you should just select Install
  3. Finish the installation normally

- Heavily reduces the ambient fog effects to increase visibility and make interiors look more natural

Wander - A Weather Mod

Installation instructions:

  1. Download the main file through MO2 and begin the installation normally
  2. You will be prompted with a FOMOD installer, in which you should select the following:
    • On the first page, select Next
    • On the second page, select Wander-lite
    • On the last page, just select Install
  3. Finish the installation normally

- Simple and lightweight weather mod that enhances and adds variety to the vanilla weather without straying too far from the vanilla feel

As visual mods are very subjective, not everyone will like Wander. If you want an alternative, I suggest checking out either Cathedral Weathers and Seasons or Azurite Weathers. They are both great mods and are fully compatible with the guide if you install one of them instead of Wander. Only one weather mod can be used at a time.
