Scrambled Bugs

Installation instructions:

  1. Download and install the Main File - Scrambled Bugs (Anniversary Edition) normally
  2. Once the file is installed, right-click on it in the left pane of MO2 and select Open in Explorer
  3. In the DLLPlugins folder, open ScrambledBugs.json in a text editor
  4. Set "underfilledSoulGems": to true (line 44)
  5. Save and exit

- Fixes many common bugs at an engine level, all fully configurable
