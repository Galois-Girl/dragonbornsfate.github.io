User Interface

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator
  2. Name the separator HUD + UI

SkyUI

- Modernizes and improves the user interface

SkyUI Flashing Savegame Fix

- Fixes the flashing savegame entries when trying to save/load a game with SkyUI

MCM Helper

Installation instructions:

  • Main File - MCM Helper SE (1.6)

- Improvements and fixes to the Mod Configuration Menu added by SkyUI

MoreHUD

Installation instructions:

  • Main File - moreHUD SE Light Master - AE

- Adds many new configurable elements to the HUD to show more useful info

TrueHUD - HUD Additions

Installation instructions:

  • Main File - TrueHUD (AE)

- Adds many configurable features to the HUD as seen in modern RPGs such as floating health bars, boss health bars, and a loot log

QUI

Installation instructions:

  • Main File - QUI-AE

- Various UI tools, tweaks, and fixes

It is highly recommended to read this mod's description to understand its features and how to take advantage of them

Contextual Crosshair

Installation instructions:

  1. Download the main file and begin the installation normally through MO2
  2. You will be prompted with a FOMOD installer, in which you should just click Install

- Adds new scripting functions for mods to take advantage of

Hide Compass Hotkey

Installation instructions:

  • Main File - hide compass hotkey 1.0

- Press a single key (X by default) to hide the compass and active quest markers

Quick Loot RE

- Adds a Fallout 4-style quick loot system

Better Dialogue Controls

- Fixes annoyances with how dialogue controls

Better MessageBox Controls

- Allows for better control over message boxes

Whose Quest Is It Anyway

- Specifies what quest an item belongs to if you try to drop it

Horse Stamina HUD

Installation instructions:

  • Main File - Horse Stamina HUD SE 1.0.3

- Displays your horse's current stamina when riding

Oxygen Meter 2

Installation instructions:

  • Main File - Oxygen Meter 2 AE
